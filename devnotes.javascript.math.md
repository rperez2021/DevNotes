## General Math Info

*Operand* - is the same as Arugement or Math Variables

*Binary* - an operator that has two operands (ex. ```y - x```)

*Unary* - an operator that has a single Operand (ex. ```-x```)

$\sqrt{4}$  can be written as ``` 4 ** (1/2) ```
