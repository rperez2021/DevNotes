## Key Value Pairs

## Example

```python
>>> myCat = {'size': 'fat', 'color': 'gray', 'disposition': 'loud'}
>>> myCat['size']
'fat'
```

Items in dictionary are unordered. And order does not matter.

Dictionary are mutable.

.keys()

.values()
